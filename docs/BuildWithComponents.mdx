# Build with components

___

Papyrum has a package that will help you document better.

#### Instalation

```
npm i @papyrum/ui-docs
```

or

```
yarn add @papyrum/ui-docs
```

### Playground

Playground for live editing react code.

Inside your `.mdx` document import `Playground` from `@papyrum/ui-docs`

```
import { Playground } from '@papyrum/ui-docs';

# My Component

<Playground>
  <div>
    Hello World, try changing the text!
  </div>
</Playground>
```

Result

<Playground>
  <div>
    Hello World, try changing the text!
  </div>
</Playground>

### Props

The Props component shows the [PropTypes](https://reactjs.org/docs/typechecking-with-proptypes.html) defined in your component within your documentation.

```
---
menu: Components
---

import { Props, Playground } from '@papyrum/ui-docs';
import { Button } from './Button';

# Button
___

### Basic usage

<Playground name="default">
  <Button>My Button</Button>
</Playground>

### Props

<Props of={Button} />
```

### Components

Contains the list of documented components, any
document that has a header `status` will be found in this list, for example:

`Button.mdx`

```
---
status: development
---

import { Playground } from '@papyrum/ui-docs';
import { Button } from './Button';

# Button

<Playground>
  <Button>My Button</Button>
</Playground>
```

Other document, `Introduction.mdx`, we import  `Components` from `@papyrum/ui-docs`.

```
import { Components } from '@papyrum/ui-docs';

### List components

<Components />
```

Result

<img src="/components.png" style={{width: '100%'}}/>

### Fonts

Component that gives you a sample of the sources
you use in your project, you only need to import `Fonts` from `@papyrum/ui-docs`.

```
import { Fonts } from '@papyrum/ui-docs';

### Fonts

<Fonts
  fontFamily='Nunito Sans'
  fonts={['34px', '24px', '14px']}
/>
```

Result

<Fonts
  fontFamily='Nunito Sans'
  fonts={['34px', '24px', '14px']}
/>

### Palette

Component that gives you a color palette simply and quickly, you just need to import
`Pallete` from `@papyrum/ui-docs`.

```
import { Palette  } from '@papyrum/ui-docs';

### Colors

<Palette colors={[
  {
    color: '#1abc9c',
    name: 'turquoise'
  },
  {
    color: '#3498db',
    name: 'blue'
  },
  {
    color: '#9b59b6',
    name: 'amethyst'
  },
  {
    color: '#e67e22',
    name: 'orange'
  }
]} />
```
