# Build with components

Papyrum has the package `@papyrum/ui-docs` that will help you document

### Playground

```
import { Playground } from '@papyrum/ui-docs';
import { Button } from './Button';

# Button

<Playground>
  <Button>My Button</Button>
</Playground>
```

In live

<Playground>
  <div>
    Hello Worldw
  </div>
</Playground>

### Props

```
import { Props } from '@papyrum/ui-docs';
import { Button } from './Button';

# Button

<Props of={Button} />
```

Result

<img src={propsImage} style={{width: '100%'}} />
