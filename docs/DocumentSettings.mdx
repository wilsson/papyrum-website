# papyrum.config.js

___

`papyrum.config.js` it contains the configuration of your application and is in the root of your directory.

all options of the `dev` and `build` commands can be used within `papyrum.config.js`

```
// papyrum.config.js
export default {
  title: 'My App',
  port: 3000,
  typescript: true,
  homepage: 'app.site',
  dest: 'dist-docs',
  ignore: 'dist',
  static: 'static'
}
```

#### `menu`

- Type: `String[]`

By default the menu items are sorted alphabetically but if you want
to change your order you can use this field indicating the names of items in the respective order.

```
// papyrum.config.js
export default {
  menu: [
    'Getting started',
    'Guides',
    'Build with components',
    'Document settings'
  ]
}
```

#### `prism`

- Type: `Object`

```
// papyrum.config.js
export default {
  prism: {
    theme: require('prism-react-renderer/themes/palenight')
  }
}
```

#### `template`

- Type: `String`

By default an index.html is created, if you want to customize the contents of `intex.html` add
this field indicating the path of your `index.html`.

```
// papyrum.config.js
export default {
  template: 'index.html'
}
```

#### `logo`

- Type: `String`

You can choose the logo of your application that is located in
sthe header with this field indicating the route or URL.

```
// papyrum.config.js
export default {
  logo: 'img/logo.png'
}
```

#### `favicon`

- Type: `String`

Route or URL of your favicon.

```
// papyrum.config.js
export default {
  logo: 'img/favicon.ico'
}
```
