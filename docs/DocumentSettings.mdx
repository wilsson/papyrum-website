# papyrum.config.js

___

`papyrum.config.js` it contains the configuration of your application and is in the root of your directory.

all options of the `dev` and `build` commands can be used within `papyrum.config.js`

```js
// papyrum.config.js
export default {
  title: 'My App',
  port: 3000,
  typescript: true,
  homepage: 'app.site',
  dest: 'dist-docs',
  ignore: 'dist',
  static: 'static'
}
```

#### `menu`

- Type: `String[]`

By default the menu items are sorted alphabetically but if you want
to change your order you can use this field indicating the names of items in the respective order.

```js
// papyrum.config.js
export default {
  menu: [
    'Getting started',
    'Guides',
    'Build with components',
    'Document settings'
  ]
}
```

#### `Colors`

Gives you the possibility to change the theme of the application for both dark mode and light mode.

By default papyrum handles the following settings

- Type: `Object`

```js
// papyrum.config.js
export default {
  colors: {
    light: {
      primary: '#01A5FE',
      header: {
        background: '#FFFFFF',
        color: '#5C6975',
      },
      sidebar: {
        background: '#FFFFFF',
        color: '#5C6975',
        activeItem: '#FFFFFF'
      },
      content: {
        background: '#FFFFFF',
        color: '#5C6975'
      }
    },
    dark: {
      primary: '#01A5FE',
      header: {
        background: '#181A1B',
        color: '#CCD5DE',
      },
      sidebar: {
        background: '#181A1B',
        color: '#CCD5DE',
        activeItem: '#FFFFFF'
      },
      content: {
        background: '#141414',
        color: '#CCD5DE'
      }
    }
  }
}
```

#### `prism`

- Type: `Object`

```js
// papyrum.config.js
export default {
  prism: {
    theme: require('prism-react-renderer/themes/palenight')
  }
}
```

#### `template`

- Type: `String`

By default an index.html is created, if you want to customize the contents of `intex.html` add
this field indicating the path of your `index.html`.

```js
// papyrum.config.js
export default {
  template: 'index.html'
}
```

#### `logo`

- Type: `String`

You can choose the logo of your application that is located in
sthe header with this field indicating the route or URL.

```js
// papyrum.config.js
export default {
  logo: 'img/logo.png'
}
```

#### `favicon`

- Type: `String`

Route or URL of your favicon.

```js
// papyrum.config.js
export default {
  logo: 'img/favicon.ico'
}
```

#### `disableShortcuts`

- Type: `Boolean`

Disable internal Shortcodes.

```js
// papyrum.config.js
export default {
  disableShortcuts: true
}
```
