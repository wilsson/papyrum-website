# Document settings

#### Name

A name that represents your document in the sidebar

```
---
name: Hello world
---
```

#### Route

```
---
route: /
---
```

#### Menu

```
---
menu: Components
---
```

#### Status

You can define the status of a component, papyrum handles 3 types of status that are:

- Development
- Deprecated
- Ready

```
---
status: development
---
```
