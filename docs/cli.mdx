# CLI

___

## Papyrum CLI commands

### `papyrum dev`

Create and serve the application with [Webpack Dev Server](https://webpack.js.org/configuration/dev-server/).

##  # `papyrum build`

Compile your application for production.


For both commands you can use the following options:

| Option | Default | Description |
| - | - | ----- |
| `--title` | name of package.json | Title of your application|
| `--port` | 3000 | Port of the development server |
| `--typescript` | false | Support for typescript |
| `--homepage` | homepage of package.json | Link from your github repository |
| `--dest` | ./.papyrum/dist | Destination location for the compilation of `papyrum build` |
| `--ignore` |[ ] | Content to ignore by papyrum, use glob matching based on [globby](https://github.com/sindresorhus/globby) |
| `--static` | static | Folder where static files will be served
